# My Website — Quick Edit Guide

## Files that matter

| File | Purpose |
|---|---|
| `index.html` | All visible content — bio, jobs, research, contact |
| `style.css` | Colors and fonts — rarely need to touch |
| `script.js` | Navigation and publications display — do not edit |
| `publications.json` | 5 most recent papers — regenerated by the script below |
| `update_publications.py` | Pulls papers from Google Scholar |
| `images/profile.png` | Your profile photo |

---

## Common edits

**Bio** — Search `EDIT: about` in `index.html`, edit the paragraphs.

**New job or degree** — Search `EDIT: experience` or `EDIT: education`. Copy an existing `<div class="timeline-item">` block, paste it above the others, and fill in your details.

**New research card** — Search `EDIT: research`. Copy a `<div class="research-card">` block. Badge options: `badge-published`, `badge-review`, `badge-prep`.

**New press mention** — Search `EDIT: news`. Copy an `<a class="news-item">` block, paste above the others (keep newest first):
```html
<a class="news-item" href="URL" target="_blank" rel="noopener">
  <span class="news-source">Source Name, YYYY</span>
  <span class="news-headline">Headline here</span>
  <i class="fa-solid fa-arrow-up-right-from-square news-icon"></i>
</a>
```

**Update publications** — Run whenever you have new papers:
```bash
pip install scholarly   # one-time setup
python update_publications.py
```

**Profile photo** — Rename your new photo to `profile.png` and drop it into the `images/` folder.

**Email or links** — Use Ctrl+H (Find & Replace) in `index.html`. Note: email is shown as `sp2596 at cornell.edu` (no `@`) to reduce spam; update both the `href="mailto:..."` and the visible `<span>` text.

---

## Publish changes

```bash
git add -A
git commit -m "Brief description"
git push
```

GitHub Pages updates automatically within ~60 seconds.
